h1. Introduction to Web Trisul 


Web Trisul is the application that allows you to visualize, search, monitor, export, and work with Trisul Network Analytics. This section introduces you to the app. Once you are familiar with navigating the application, accessing the analysis tools, and basic administration tasks, you should be easily able to intuitively accomplish most of the tasks.
